[English](README.md)

고품질의 접근성 높은 웹 앱과 디자인 시스템을 구축하기 위한 오픈소스 UI 컴포넌트 라이브러리입니다.

Vapor는 React 기반의 UI 라이브러리로, 접근성, 커스터마이징, 그리고 개발자 경험에 중점을 두고 설계되었습니다. 여러분의 디자인 시스템을 위한 기반 레이어로 사용하거나 기존 프로젝트에 점진적으로 도입할 수 있습니다.

## 패키지

- [`@vapor-ui/core`](https://vapor-ui.goorm.io/docs/getting-started/installation) – 핵심 컴포넌트 라이브러리 – 34개 이상의 접근성 높은 컴포넌트
- [`@vapor-ui/hooks`](https://www.npmjs.com/package/@vapor-ui/hooks) – 상태 및 UI 관리를 위한 재사용 가능한 React 훅 모음
- [`@vapor-ui/icons`](https://www.npmjs.com/package/@vapor-ui/icons) – SVG 아이콘 React 컴포넌트 세트
- [`@vapor-ui/codemod`](https://www.npmjs.com/package/@vapor-ui/codemod) – 안전한 컴포넌트 업데이트를 위한 자동 코드 마이그레이션 도구
- [`@vapor-ui/color-generator`](https://www.npmjs.com/package/@vapor-ui/color-generator) – Adobe Leonardo 기반 WCAG 호환 컬러 팔레트 생성기
- [`@vapor-ui/css-generator`](https://www.npmjs.com/package/@vapor-ui/css-generator) – Vapor UI 디자인 시스템 테마용 CSS 변수 생성기

## 시작하기

```bash
npm i @vapor-ui/core @vapor-ui/icons @vapor-ui/hooks
```

자세한 사용법은 [공식 문서](https://vapor-ui.goorm.io/docs/getting-started/installation)를 참고하세요.
