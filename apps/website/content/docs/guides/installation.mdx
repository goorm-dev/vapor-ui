### 1. Install the package

Install Vapor UI using your preferred package manager:

```package-install
npm install @vapor-ui/core
```

<br />

### 2. Add styles import

Add styles import to the root of your application:

{/* prettier-ignore */}
<Tabs items={[".jsx", ".css"]} defaultValue=".jsx">
  <Tab value=".jsx">
  ```tsx
  import '@vapor-ui/core/styles.css'; 
  ```
  </Tab>
  <Tab value=".css">
  ```css
  @import '@vapor-ui/core/styles.css'; 
  ```
  </Tab>
</Tabs>

<br />

### 3. Assemble a component

You can now start using Vapor UI components in your React application. Here's an example of how to assemble a simple component:

<Demo name="assemble-component">
```tsx
import { Card } from "@vapor-ui/core";

export default function AssembleComponent() {
  return (
    <Card>
      <Card.Header>
        <h2>Assemble Component</h2>
      </Card.Header>
      <Card.Body>
        <p>This is an example of assembling a component using Vapor UI.</p>
      </Card.Body>
    </Card>
  );
}
```
</Demo>
