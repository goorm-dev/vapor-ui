그룹이 통합하는 시각(테두리·배경 제거, 높이 정렬)을 적용받으려면 입력과 버튼을 각각 `InputGroup.Input`과 `InputGroup.Button`으로 감싸야 합니다. 래퍼 없이 `TextInput`이나 `IconButton` 같은 컨트롤을 그룹 안에 직접 넣으면 그룹이 스타일을 통합하지 않으며, 이때는 각 컨트롤의 시각을 직접 맞춰야 합니다.
