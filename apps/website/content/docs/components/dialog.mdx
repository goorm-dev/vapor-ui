## Examples

### Simple Usage

기본적인 Dialog 사용법입니다.

<Demo name="dialog-simple">
```tsx
import { Dialog, Button } from '@vapor-ui/core';

export default function DialogSimple() {
    return (
        <Dialog.Root>
            <Dialog.Trigger asChild>
                <Button>단순 다이얼로그 열기</Button>
            </Dialog.Trigger>
            <Dialog.CombinedContent>
                <Dialog.Header>
                    <Dialog.Title>알림</Dialog.Title>
                    <Dialog.Close aria-label="Close" />
                </Dialog.Header>
                <Dialog.Body>
                    <Dialog.Description>
                        이것은 기본적인 다이얼로그입니다. 
                        Escape 키나 배경 클릭으로 닫을 수 있습니다.
                    </Dialog.Description>
                </Dialog.Body>
                <Dialog.Footer>
                    <Dialog.Close asChild>
                        <Button>확인</Button>
                    </Dialog.Close>
                </Dialog.Footer>
            </Dialog.CombinedContent>
        </Dialog.Root>
    );
}
```
</Demo>

### Composition Pattern

Dialog.Root, Dialog.Trigger, Dialog.CombinedContent, Dialog.Header, Dialog.Body, Dialog.Footer를 조합한 패턴입니다.

<Demo name="dialog-composition">
```tsx
import { Dialog, Button } from '@vapor-ui/core';

export default function DialogComposition() {
    return (
        <Dialog.Root size="lg" closeOnClickOverlay={false}>
            <Dialog.Trigger asChild>
                <Button variant="outline">확인 다이얼로그</Button>
            </Dialog.Trigger>
            <Dialog.CombinedContent>
                <Dialog.Header>
                    <Dialog.Title>작업 확인</Dialog.Title>
                    <Dialog.Close aria-label="Close" />
                </Dialog.Header>
                <Dialog.Body>
                    <Dialog.Description>
                        이 작업을 진행하시겠습니까? 
                        이 작업은 되돌릴 수 없습니다.
                    </Dialog.Description>
                </Dialog.Body>
                <Dialog.Footer>
                    <Dialog.Close asChild>
                        <Button variant="ghost">취소</Button>
                    </Dialog.Close>
                    <Dialog.Close asChild>
                        <Button variant="danger">삭제</Button>
                    </Dialog.Close>
                </Dialog.Footer>
            </Dialog.CombinedContent>
        </Dialog.Root>
    );
}
```
</Demo>

## Props Table

### Dialog.Root

Dialog의 메인 컨테이너 컴포넌트입니다. 다이얼로그의 크기, 모달 동작, 열림/닫힘 상태 등의 속성을 제공하며, 모든 하위 컴포넌트에서 공통으로 사용할 수 있도록 컨텍스트를 제공합니다.

<ComponentPropsTable file="dialog" section="props" />

### Dialog.Trigger

Dialog를 열기 위한 트리거 컴포넌트입니다. 일반적으로 버튼으로 사용되며, asChild prop을 통해 다른 컴포넌트를 트리거로 사용할 수 있습니다.

<ComponentPropsTable file="dialog" section="triggerProps" />

### Dialog.Content

Dialog의 콘텐츠 컨테이너 컴포넌트입니다. 다이얼로그 외부 클릭이나 키보드 이벤트 등의 세부적인 동작을 제어할 수 있습니다.
