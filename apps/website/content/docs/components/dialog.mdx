## Props Table

---

### Dialog.Root

Dialog의 하위 요소를 그룹화합니다. 자체 HTML 요소를 렌더링하지 않습니다.

<ComponentPropsTable file="dialog" section="props" />

### Dialog.PortalPrimitive

Dialog 콘텐츠를 document.body에 렌더링합니다.

<ComponentPropsTable file="dialog" section="portalPrimitiveProps" />

### Dialog.OverlayPrimitive

Dialog의 배경 오버레이를 표시합니다. `<div>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="overlayPrimitiveProps" />

### Dialog.PopupPrimitive

Dialog의 콘텐츠 영역입니다. `<div>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="popupPrimitiveProps" />

### Dialog.Popup

Dialog의 컨텐츠를 표시합니다.
`PortalPrimitive`, `OverlayPrimitive`, `PopupPrimitive`가 통합되어 있습니다.

<ComponentPropsTable file="dialog" section="popupProps" />

### Dialog.Trigger

Dialog를 여는 트리거 버튼입니다. `<button>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="triggerProps" />

### Dialog.Close

Dialog를 닫는 버튼입니다. `<button>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="closeProps" />

### Dialog.Title

Dialog의 제목을 표시합니다. `<h2>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="titleProps" />

### Dialog.Description

Dialog의 설명 텍스트를 표시합니다. `<p>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="descriptionProps" />

### Dialog.Header

Dialog의 상단 영역을 나타냅니다. `<div>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="headerProps" />

### Dialog.Body

Dialog의 주요 콘텐츠가 포함됩니다. `<div>` 요소를 렌더링합니다.

<ComponentPropsTable file="dialog" section="bodyProps" />

### Dialog.Footer
