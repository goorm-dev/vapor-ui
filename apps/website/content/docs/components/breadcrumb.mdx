## Props Table

---

### Breadcrumb.Root

Breadcrumb의 하위 요소를 그룹화합니다. `<nav>` 요소를 렌더링하며, 내부 상태를 공유합니다.

<ComponentPropsTable file="breadcrumb" section="props" />

### Breadcrumb.Item

각각의 Breadcrumb 항목입니다. `<li>`와 `<a>`의 중첩 구조로 이루어져 있습니다.

<ComponentPropsTable file="breadcrumb" section="itemProps" />

### Breadcrumb.Separator

Breadcrumb Item을 구분하는 구분자입니다. `<li>`와 구분자 아이콘으로 이루어져 있습니다.

<ComponentPropsTable file="breadcrumb" section="separatorProps" />

### Breadcrumb.Ellipsis

다수의 Breadcrumb 항목을 생략하는 요소입니다. `<li>`와 말줄임표 아이콘으로 이루어져 있습니다.

<ComponentPropsTable file="breadcrumb" section="ellipsisProps" />

### Breadcrumb.ListPrimitive

Breadcrumb 항목들을 담는 순서가 있는 리스트 컨테이너입니다. `<ol>` 요소를 렌더링 합니다.

<ComponentPropsTable file="breadcrumb" section="listPrimitiveProps" />

### Breadcrumb.LinkPrimitive

Breadcrumb 내의 클릭 가능한 링크 컴포넌트입니다. `<a>` 요소를 렌더링 합니다.

<ComponentPropsTable file="breadcrumb" section="linkPrimitiveProps" />
