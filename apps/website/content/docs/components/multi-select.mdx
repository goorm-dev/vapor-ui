선택된 값들을 표시하는 컴포넌트입니다. 기본적으로 배지 형태로 표시되며, 함수형 children을 통해 커스텀 값 표시가 가능합니다.

<ComponentPropsTable file="multi-select" section="valuePrimitiveProps" />
