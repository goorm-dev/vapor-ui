### Using Tabs.Panel

`Tabs.Panel`을 사용하여 각 탭과 연결된 콘텐츠를 표시할 수 있습니다. Panel은 선택사항이며, 필요에 따라 생략할 수 있습니다.

```tsx
<Tabs.Root defaultValue="profile">
    <Tabs.List>
        <Tabs.Button value="profile">프로필</Tabs.Button>
        <Tabs.Button value="settings">설정</Tabs.Button>
    </Tabs.List>

    <Tabs.Panel value="profile">
        <ProfileContent />
    </Tabs.Panel>

    <Tabs.Panel value="settings">
        <SettingsContent />
    </Tabs.Panel>
</Tabs.Root>
```

### Custom Indicator

`Tabs.List`는 기본적으로 `Tabs.IndicatorPrimitive`를 포함합니다. Indicator를 커스터마이징하려면 `indicatorElement` prop을 사용하세요.

### Tabs.ListPrimitive와 Tabs.IndicatorPrimitive

더 세밀한 제어가 필요한 경우 `Tabs.ListPrimitive`와 `Tabs.IndicatorPrimitive`를 직접 조합할 수 있습니다.

```tsx
<Tabs.Root defaultValue="home">
    <Tabs.ListPrimitive>
        <Tabs.Button value="home">홈</Tabs.Button>
        <Tabs.Button value="about">소개</Tabs.Button>
        <Tabs.IndicatorPrimitive className="custom-indicator" />
    </Tabs.ListPrimitive>
</Tabs.Root>
```
