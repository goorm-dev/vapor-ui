## Property

---

### Size

Textarea 크기는 sm, md, lg, xl 로 제공합니다.

<Demo name="textarea-size">
```tsx
import { Textarea } from '@vapor-ui/core';

export default function TextareaSize() {
    return (
        <div className="space-y-4">
            <Textarea.Root size="sm" placeholder="Small size">
                <Textarea.Input />
            </Textarea.Root>
            <Textarea.Root size="md" placeholder="Medium size">
                <Textarea.Input />
            </Textarea.Root>
            <Textarea.Root size="lg" placeholder="Large size">
                <Textarea.Input />
            </Textarea.Root>
            <Textarea.Root size="xl" placeholder="Extra large size">
                <Textarea.Input />
            </Textarea.Root>
        </div>
    );
}
```

</Demo>

### Auto Resize

텍스트 내용에 따라 자동으로 높이가 조절되는 기능을 제공합니다.

<Demo name="textarea-auto-resize">
```tsx
import { Box, Textarea } from '@vapor-ui/core';

export default function TextareaAutoResize() {
    return (
        <Box display="flex" alignItems="center" justifyContent="center">
            <Textarea.Root
                autoResize={true}
                placeholder="텍스트를 입력하면 자동으로 높이가 조절됩니다..."
            >
                <Textarea.Input />
            </Textarea.Root>
        </Box>
    );
}
```
</Demo>

### Resizing

사용자가 직접 크기를 조절할 수 있는 기능을 제공합니다.

<Demo name="textarea-resizing">
```tsx
import { Box, Textarea } from '@vapor-ui/core';

export default function TextareaResizing() {
    return (
        <Box>
            <Textarea.Root
                resizing={true}
                placeholder="크기 조절 가능 (우하단 모서리를 드래그하세요)"
            >
                <Textarea.Input />
            </Textarea.Root>
        </Box>
    );
}
```

</Demo>

Textarea의 다양한 상태를 보여줍니다.

<Demo name="textarea-states">
```tsx

                <Textarea.Root disabled placeholder="Disabled state">
                    <Textarea.Input />
                </Textarea.Root>
            </Grid.Item>
            <Grid.Item>
                <Textarea.Root invalid placeholder="Invalid state">
                    <Textarea.Input />
                </Textarea.Root>
            </Grid.Item>
            <Grid.Item>
                <Textarea.Root readOnly defaultValue="Read only content">
                    <Textarea.Input />

```

</Demo>
