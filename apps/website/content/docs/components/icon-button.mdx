## Props Table

---

IconButton은 아이콘만으로 작업을 수행하는 버튼 컴포넌트입니다. Button 컴포넌트를 확장하여 추가적인 shape 속성과 아이콘 전용 스타일링을 제공합니다. 접근성을 위해 적절한 `aria-label`을 반드시 제공해야 합니다. 이 컴포넌트는 `button` 요소를 기반으로 합니다.
