## Props Table

### RadioGroup.Root

RadioGroup의 메인 컨테이너 컴포넌트입니다. 라디오 버튼 그룹의 크기, 방향, 상태 등의 속성을 제공하며, 모든 하위 컴포넌트에서 공통으로 사용할 수 있도록 컨텍스트를 제공합니다.

<ComponentPropsTable file="radio-group" section="props" />

### RadioGroup.Item

RadioGroup의 개별 아이템 컨테이너 컴포넌트입니다. 각각의 라디오 버튼과 레이블을 감싸는 역할을 하며, 아이템별 상태 관리를 담당합니다.

<ComponentPropsTable file="radio-group" section="itemProps" />

### RadioGroup.Control

RadioGroup의 실제 라디오 버튼 컴포넌트입니다. 사용자가 클릭할 수 있는 원형 버튼을 표시하며, 선택 상태에 따라 시각적 피드백을 제공합니다.

<ComponentPropsTable file="radio-group" section="controlProps" />

### RadioGroup.Label

RadioGroup의 레이블 컴포넌트입니다. 각 라디오 버튼 옵션의 설명 텍스트를 표시하며, 접근성을 위해 해당 Control과 자동으로 연결됩니다.

<ComponentPropsTable file="radio-group" section="labelProps" />
