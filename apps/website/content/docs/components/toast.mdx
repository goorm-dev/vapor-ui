Toast 컴포넌트는 화면 모서리에 일시적인 알림 메시지를 표시합니다.

사용자 액션에 대한 피드백을 제공하며, 알림을 단일 목록으로 관리하여 다른 컴포넌트와 사용 방법이 다릅니다.
