## Settings

---

### Node.js

라이브러리를 사용하기 위해 v16 이상의 Node.js 버전이 필요합니다.

## Library Installation

---

#### Peer dependencies

Vapor Core를 설치하기 전에 다음의 하위 의존성 패키지를 설치해야 합니다.

```json title="package.json"
"dependencies": {
  "react": ">=18",
  "react-dom": ">=18",
},
```
