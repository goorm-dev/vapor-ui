## Vite - React

Vite는 빠른 개발 서버와 최적화된 빌드 성능을 제공하는 차세대 프론트엔드 도구입니다. Vite 환경에서는 별도의 설정 없이도 `@vapor-ui/core`의 **Tree-shaking이 기본적으로 지원**됩니다.

<Steps>
<Step>

### 1단계: 프로젝트 생성

TypeScript를 사용하는 새로운 Vite React 프로젝트를 생성합니다.

```bash
npm create vite@latest your-vite-project -- --template react-ts
```

</Step>

<Step>

### 2단계: 라이브러리 설치

생성된 프로젝트 디렉토리로 이동하여 `@vapor-ui/core`를 설치합니다.

```bash
cd your-vite-project
npm install
```

```package-install
npm install @vapor-ui/core @vapor-ui/icons
```

</Step>

<Step>

### 3단계: 컴포넌트 사용

`src/App.tsx` 파일에서 컴포넌트를 가져와 사용합니다.

```tsx title="src/App.tsx"
import { Button } from '@vapor-ui/core';

import './App.css';

function App() {
    return (
        <div className="App">
            <h1>Vite + React with @vapor-ui/core</h1>
            <Button>Hello, World!</Button>
        </div>
    );
}

export default App;
```

</Step>
</Steps>
