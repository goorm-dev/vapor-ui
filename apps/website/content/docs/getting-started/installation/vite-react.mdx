이 가이드는 Vite 7.1.0, React 19.1.1, TypeScript 5.8.3 환경에서 Vapor UI를 설정하고 사용하는 방법을 단계별로 안내합니다. Vite는 최소한의 설정으로 최대의 개발자 경험을 제공합니다.

## 🚀 Vapor UI & Vite + React 프로젝트 설정 가이드

<Steps>
<Step>

### 1단계: 프로젝트 생성 및 의존성 설치

최신 Vite와 React를 사용한 프로젝트를 생성하고 필요한 라이브러리를 설치합니다.

```bash
# 1. Vite React 프로젝트 생성 (TypeScript 템플릿)
npm create vite@latest my-vapor-vite-app -- --template react-ts

# 2. 프로젝트 폴더로 이동
cd my-vapor-vite-app

# 3. 기본 의존성 설치
npm install
```

```package-install
npm install @vapor-ui/core@^0.4.0 @vapor-ui/icons
```

**개발 도구 설치 (선택사항):**
```bash
npm install --save-dev rollup-plugin-visualizer
```

<Step>

### 2단계: Vite 설정

번들 분석을 위한 간단한 Vite 설정을 구성합니다.

```typescript title="vite.config.ts"
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { visualizer } from 'rollup-plugin-visualizer';

export default defineConfig({
  plugins: [
    react(),
    // 번들 분석을 위한 플러그인 (선택사항)
    visualizer({
      filename: 'dist/stats.html',
      open: true,
      gzipSize: true,
      brotliSize: true,
    }),
  ],
  
  // 개발 서버 설정
  server: {
    port: 3000,
    open: true,
  },
  
  // 빌드 최적화
  build: {
    rollupOptions: {
      output: {
        manualChunks: {
          vendor: ['react', 'react-dom'],
          'vapor-ui': ['@vapor-ui/core', '@vapor-ui/icons'],
        },
      },
    },
  },
});
```

</Step>

<Step>

### 3단계: FOUC 방지 설정

테마 깜빡임을 방지하기 위해 `index.html`에 인라인 스크립트를 추가합니다.

```html title="index.html"
<!doctype html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <link rel="icon" type="image/svg+xml" href="/vite.svg" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Vite + React + Vapor UI</title>
    
    <!-- FOUC 방지를 위한 테마 초기화 스크립트 -->
    <script>
      (function() {
        try {
          const defaultConfig = { 
            appearance: "light", 
            radius: "md", 
            scaling: 1.0,
            primaryColor: "#ffc107"
          };
          
          // LocalStorage에서 테마 설정 로드
          const stored = localStorage.getItem('my-vite-app-theme');
          const config = stored ? JSON.parse(stored) : defaultConfig;
          
          const root = document.documentElement;
          
          // 테마 클래스 적용
          root.classList.add(`vapor-${config.appearance}-theme`);
          
          // CSS 변수 설정
          const radiusFactors = {
            'none': '0', 'sm': '0.125', 'md': '0.25', 
            'lg': '0.5', 'xl': '0.75', 'full': '9999'
          };
          root.style.setProperty('--vapor-radius-factor', radiusFactors[config.radius] || '0.25');
          root.style.setProperty('--vapor-scaling-factor', config.scaling.toString());
          root.style.setProperty('--vapor-primary-color', config.primaryColor);
        } catch (e) {
          console.warn('테마 초기화 실패:', e);
        }
      })();
    </script>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.tsx"></script>
  </body>
</html>
```

</Step>

### 4단계: 테마 설정

커스텀 테마 설정 파일을 생성합니다.

```typescript title="src/lib/theme.config.ts"
import { createThemeConfig } from '@vapor-ui/core';

export const vaporTheme = createThemeConfig({
  appearance: 'light',
  primaryColor: '#ffc107', // Amber/Yellow 색상
  scaling: 1.0,
  radius: 'md',
  storageKey: 'my-vite-app-theme',
});

// 테마 타입 정의
export interface AppThemeConfig {
  appearance: 'light' | 'dark';
  radius: 'none' | 'sm' | 'md' | 'lg' | 'xl' | 'full';
  scaling: number;
  primaryColor: string;
}

export const defaultTheme: AppThemeConfig = {
  appearance: 'light',
  radius: 'md',
  scaling: 1.0,
  primaryColor: '#ffc107',
};
```

</Step>

### 5단계: 메인 앱 설정

`src/main.tsx`에서 ThemeProvider로 앱을 래핑합니다.

```tsx title="src/main.tsx"
import { StrictMode } from 'react';
import { createRoot } from 'react-dom/client';
import { ThemeProvider } from '@vapor-ui/core';
import { vaporTheme } from './lib/theme.config';

// Vapor UI 스타일 import
import '@vapor-ui/core/styles.css';
import './index.css';

import App from './App.tsx';

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <ThemeProvider config={vaporTheme}>
      <App />
    </ThemeProvider>
  </StrictMode>,
);
```

### 7단계: 포괄적인 데모 앱 구성

다양한 컴포넌트와 테마 제어 기능을 포함한 메인 앱을 만듭니다.

```tsx title="src/App.tsx"
import { useState } from 'react';
import {
  Button,
  Card,
  Text,
  Badge,
  Switch,
  Box,
  Grid,
  useTheme,
} from '@vapor-ui/core';
import { DarkIcon, LightIcon, SettingsIcon } from '@vapor-ui/icons';
import './App.css';

function App() {
  const { appearance, radius, scaling, primaryColor, setTheme } = useTheme();
  const [showSettings, setShowSettings] = useState(false);

  const toggleTheme = () => {
    setTheme({
      appearance: appearance === 'dark' ? 'light' : 'dark',
    });
  };

  const radiusOptions = ['none', 'sm', 'md', 'lg', 'xl', 'full'] as const;
  const buttonVariants = ['primary', 'secondary', 'outline', 'ghost', 'fill'] as const;

  return (
    <Box className="min-h-screen p-8">
      <div className="max-w-6xl mx-auto">
        {/* 헤더 */}
        <div className="mb-8 text-center">
          <Text asChild typography="heading1" className="mb-4">
            <h1>Vite + React + Vapor UI</h1>
          </Text>
          <Text className="mb-6">
            Vite 7.1.0, React 19.1.1 환경에서의 포괄적인 컴포넌트 데모
          </Text>
          
          <div className="flex justify-center gap-2 mb-4">
            <Button onClick={toggleTheme} color="primary">
              {appearance === 'dark' ? <LightIcon /> : <DarkIcon />}
              {appearance === 'dark' ? '라이트 모드' : '다크 모드'}
            </Button>
            
            <Button 
              onClick={() => setShowSettings(!showSettings)}
              variant="outline"
            >
              <SettingsIcon />
              설정
            </Button>
          </div>

          {/* 현재 테마 상태 표시 */}
          <div className="flex justify-center gap-2">
            <Badge color="primary">Appearance: {appearance}</Badge>
            <Badge color="secondary">Radius: {radius}</Badge>
            <Badge variant="outline">Scaling: {scaling}</Badge>
          </div>
        </div>

        {/* 설정 패널 */}
        {showSettings && (
          <Card.Root className="mb-8">
            <Card.Header>테마 설정</Card.Header>
            <Card.Body>
              <Grid cols={{ initial: 1, md: 2 }} gap="6">
                <div>
                  <Text className="mb-2 font-semibold">Radius 설정</Text>
                  <div className="flex flex-wrap gap-2">
                    {radiusOptions.map((r) => (
                      <Button
                        key={r}
                        size="sm"
                        variant={radius === r ? 'primary' : 'outline'}
                        onClick={() => setTheme({ radius: r })}
                      >
                        {r}
                      </Button>
                    ))}
                  </div>
                </div>
                
                <div>
                  <Text className="mb-2 font-semibold">Scaling 설정</Text>
                  <div className="flex gap-2">
                    {[0.8, 0.9, 1.0, 1.1, 1.2].map((s) => (
                      <Button
                        key={s}
                        size="sm"
                        variant={scaling === s ? 'primary' : 'outline'}
                        onClick={() => setTheme({ scaling: s })}
                      >
                        {s}x
                      </Button>
                    ))}
                  </div>
                </div>
              </Grid>
            </Card.Body>
          </Card.Root>
        )}

        {/* 컴포넌트 데모 */}
        <Grid cols={{ initial: 1, md: 2, lg: 3 }} gap="6">
          {/* 버튼 변형 */}
          <Card.Root>
            <Card.Header>버튼 변형</Card.Header>
            <Card.Body className="space-y-3">
              {buttonVariants.map((variant) => (
                <Button key={variant} variant={variant} className="w-full">
                  {variant.charAt(0).toUpperCase() + variant.slice(1)}
                </Button>
              ))}
            </Card.Body>
          </Card.Root>

          {/* 인터랙티브 요소 */}
          <Card.Root>
            <Card.Header>인터랙티브 요소</Card.Header>
            <Card.Body className="space-y-4">
              <div className="flex items-center justify-between">
                <Text>다크 모드</Text>
                <Switch 
                  checked={appearance === 'dark'}
                  onCheckedChange={() => toggleTheme()}
                />
              </div>
              <div className="flex gap-2">
                <Badge color="primary">Primary</Badge>
                <Badge color="secondary">Secondary</Badge>
                <Badge variant="outline">Outline</Badge>
              </div>
            </Card.Body>
          </Card.Root>

          {/* 시스템 정보 */}
          <Card.Root>
            <Card.Header>시스템 정보</Card.Header>
            <Card.Body className="space-y-2">
              <Text size="sm">Primary Color: {primaryColor}</Text>
              <Text size="sm">현재 테마: {appearance}</Text>
              <Text size="sm">Radius: {radius}</Text>
              <Text size="sm">Scaling: {scaling}x</Text>
            </Card.Body>
          </Card.Root>
        </Grid>
      </div>
    </Box>
  );
}

export default App;
```

## 주요 장점

### 최소한의 설정
- **Zero Config**: 복잡한 빌드 설정 불필요
- **자동 Tree-shaking**: Vite가 기본적으로 지원
- **즉시 시작**: 설정 완료 후 바로 개발 가능

### 뛰어난 개발자 경험
- **즉시 HMR**: 코드 변경 시 즉각적인 업데이트
- **빠른 빌드**: 개발 서버 즉시 시작 (Cold Start ~200ms)
- **번들 분석**: 내장된 성능 모니터링

### 고급 테마 시스템
- **실시간 테마 전환**: 페이지 새로고침 없이 테마 변경
- **CSS 변수 기반**: 스케일링 및 반지름 실시간 조정
- **LocalStorage 지속성**: 세션 간 테마 설정 유지
- **FOUC 방지**: 인라인 스크립트로 깜빡임 현상 제거

### 성능 최적화
- **청크 분할**: vendor, vapor-ui 별도 번들링
- **압축 분석**: Gzip, Brotli 압축 크기 모니터링
- **개발 도구**: 번들 크기 시각화 및 최적화 가이드
