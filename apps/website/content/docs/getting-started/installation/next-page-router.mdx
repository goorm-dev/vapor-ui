## Vapor UI & Next.js Pages Router 프로젝트 설정 가이드

<Steps>
<Step>

### 1단계: 프로젝트 생성 및 의존성 설치

Next.js 프로젝트를 생성하고 필요한 라이브러리를 설치합니다.

```bash
# 1. Next.js Pages Router 프로젝트 생성 (TypeScript 사용)
npx create-next-app@latest my-vapor-pages-app --typescript

# 2. 프로젝트 폴더로 이동
cd my-vapor-pages-app
```

```package-install
npm install @vapor-ui/core@latest @vapor-ui/icons@latest
```

</Step>

<Step>

### 2단계: Next.js 설정

Pages Router에 특화된 설정으로 `next.config.mjs` 파일을 구성합니다.

```javascript title="next.config.ts"
import type { NextConfig } from "next";

const nextConfig: NextConfig = {
  reactStrictMode: true,
  transpilePackages: ["@vapor-ui/core", "@vapor-ui/icons"],
};

export default nextConfig;
```

</Step>

<Step>

### 3단계: CSS 아키텍처 설정

CSS 레이어 구조를 설정하여 스타일 우선순위를 정의합니다.

```css title="styles/globals.css"
/* CSS 레이어 우선순위 정의 */
@layer reset, tw-base, base, tw-theme, theme, components, utilities, tw-utilities;

@import '@vapor-ui/core/styles.css';

@import 'tailwindcss/preflight' layer(tw-base);
@import 'tailwindcss/theme' layer(tw-theme);
@import 'tailwindcss/utilities' layer(tw-utilities);

:root {
    --background: #ffffff;
    --foreground: #171717;
}

@theme inline {
    --color-background: var(--background);
    --color-foreground: var(--foreground);
    --font-sans: var(--font-geist-sans);
    --font-mono: var(--font-geist-mono);
}

@media (prefers-color-scheme: dark) {
    :root {
        --background: #0a0a0a;
        --foreground: #ededed;
    }
}

body {
    background: var(--background);
    color: var(--foreground);
    font-family: Arial, Helvetica, sans-serif;
}
```

</Step>

<Step>

### 4단계: 테마 설정

테마 관리 시스템을 구성합니다.

```typescript title="lib/theme.config.ts"
import { createThemeConfig } from '@vapor-ui/core';

export const vaporTheme = createThemeConfig({
    appearance: 'light',
    primaryColor: '#0066ff',
    scaling: 1,
    radius: 'md',
});
```

</Step>

<Step>

### 5단계: 글로벌 앱 설정 (\_app.tsx)

Pages Router의 핵심인 `_app.tsx`에서 전역 설정을 구성합니다.

```tsx title="pages/_app.tsx"
import { ThemeProvider } from '@vapor-ui/core';
import type { AppProps } from 'next/app';

import { vaporTheme } from '../lib/theme.config';

import '../styles/globals.css';

export default function App({ Component, pageProps }: AppProps) {
    return (
        <ThemeProvider config={vaporTheme}>
            <Component {...pageProps} />
        </ThemeProvider>
    );
}
```

</Step>

<Step>

### 6단계: 문서 구조 설정 (\_document.tsx)

FOUC 방지를 위한 문서 구조를 설정합니다.

```tsx title="pages/_document.tsx"
import { ThemeScript } from '@vapor-ui/core';
import { Head, Html, Main, NextScript } from 'next/document';

export default function Document() {
    return (
        <Html lang="ko">
            <Head>
                {/* FOUC(깜빡임) 방지를 위한 스크립트 */}
                <ThemeScript />
            </Head>
            <body>
                <Main />
                <NextScript />
            </body>
        </Html>
    );
}
```

</Step>

<Step>

### 7단계: 메인 페이지 구성

포괄적인 컴포넌트 쇼케이스를 포함한 메인 페이지를 만듭니다.

```tsx title="pages/index.tsx"
'use client';

// 클라이언트 컴포넌트로 선언
import { Button, Card, CardBody, Text, useTheme } from '@vapor-ui/core';
import { DarkIcon, LightIcon } from '@vapor-ui/icons';

export default function HomePage() {
    const { appearance, setTheme } = useTheme();

    return (
        <main className="flex min-h-screen flex-col items-center justify-center p-24">
            <Card.Root className="max-w-md">
                <CardBody>
                    <div className="flex flex-col gap-4">
                        <Text asChild typography="heading1" foreground="normal">
                            <h1>Welcome to Vapor UI!</h1>
                        </Text>
                        <Text>
                            This is an example of how to use Vapor UI components with Next.js.
                        </Text>

                        {/* 버튼 컴포넌트 (다양한 variant) */}
                        <div className="flex items-center gap-2">
                            <Button color="primary">Primary</Button>
                            <Button variant="ghost">Ghost</Button>
                        </div>

                        {/* 테마 전환 버튼 */}
                        <Button
                            aria-label="Toggle theme"
                            onClick={() => {
                                setTheme({
                                    appearance: appearance === 'dark' ? 'light' : 'dark',
                                });
                            }}
                            color="primary"
                        >
                            {appearance === 'dark' ? <LightIcon /> : <DarkIcon />}
                        </Button>
                    </div>
                </CardBody>
            </Card.Root>
        </main>
    );
}
```

</Step>
</Steps>
