# @vapor-ui/icons

A set of SVG icon React components for Vapor UI.

## Installation

```sh
# pnpm
pnpm add @vapor-ui/icons

# npm
npm install @vapor-ui/icons

# yarn
yarn add @vapor-ui/icons
```

> For detailed usage instructions, please refer to the [official documentation](https://vapor-ui.goorm.io/docs/icons).
