An open-source UI component library for building high-quality, accessible web apps and design systems.

Vapor is a React-based UI library designed with a focus on accessibility, customization, and developer experience. Use it as a foundation layer for your design system or gradually adopt it into existing projects.

## Packages

- [`@vapor-ui/core`](https://vapor-ui.goorm.io/docs/getting-started/installation) – core components library – 34+ accessible components
- [`@vapor-ui/hooks`](https://www.npmjs.com/package/@vapor-ui/hooks) – collection of reusable React hooks for state and UI management
- [`@vapor-ui/icons`](https://www.npmjs.com/package/@vapor-ui/icons) – a set of SVG icon React components
- [`@vapor-ui/codemod`](https://www.npmjs.com/package/@vapor-ui/codemod) – automated code migration tools for safe component updates
- [`@vapor-ui/color-generator`](https://www.npmjs.com/package/@vapor-ui/color-generator) – WCAG-compliant color palette generator built on Adobe Leonardo
- [`@vapor-ui/css-generator`](https://www.npmjs.com/package/@vapor-ui/css-generator) – generate CSS variables for Vapor UI design system themes

## Getting Started

```bash
npm i @vapor-ui/core @vapor-ui/icons @vapor-ui/hooks
```

For detailed usage instructions, please refer to the [official documentation](https://vapor-ui.goorm.io/docs/getting-started/installation).
